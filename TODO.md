### 4. THREE.Clock Deprecation Warning in Console
**File:** `src/lib/three/HeroScene.tsx`
**Issue:** Browser console shows: `THREE.THREE.Clock: This module has been deprecated. Please use THREE.Timer instead.`
This may cause animation issues in future Three.js versions.
**Fix:**
- Replace any `new THREE.Clock()` usage with `new THREE.Timer()` in HeroScene.tsx.
- Or use `@react-three/fiber`'s built-in `useFrame` delta parameter instead of a manual Clock.

## 🟡 P2 — Medium Visual Issues (Fix After P1)

### 5. Results Section — Uneven Topper Card Grid (3+2 Layout)
**File:** `src/components/sections/Results.tsx`
**Issue:** With 5 topper cards, the grid renders as 3 cards in row 1 and 2 cards in row 2.
The bottom row looks visually unbalanced (two cards with large empty space on right).
**Fix (Option A):** Use `grid-cols-2 md:grid-cols-3` — this gives 3+2 layout but is acceptable if cards are centered. Add `justify-center` to the grid container.
**Fix (Option B — Preferred):** Add a 6th topper card to `en.json` + `hi.json` so the grid is 3+3.
**Fix (Option C):** Use `grid-cols-2 lg:grid-cols-5` for a single even row on large screens.
- In all cases, add `justify-center` or `mx-auto` to the grid so partial rows are centered.

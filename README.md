### Vertical Spacing System

Spacing between every sibling inside `.layout-container` is driven by one CSS custom property:

```css
:root        { --vgap: 2.5rem; }
@media ≥640px  { --vgap: 3.5rem; }
@media ≥1024px { --vgap: 5rem;   }

.layout-container > * + * { margin-top: var(--vgap); }
```

**Never add manual `mb-*` or `mt-*` to direct children of `.layout-container`.** The CSS rule handles all of them symmetrically.

### Section Padding (`section-pad`)

```css
mobile   : 6rem top + 6rem bottom
≥640px   : 8rem  top + 8rem bottom
≥768px   : 11rem top + 11rem bottom
≥1024px  : 13rem top + 13rem bottom
```

## ⚡ Hooks Reference

### `useScrollReveal(ref, options)`
GSAP ScrollTrigger hook. Animates `.gsap-reveal` children of the container ref.
```tsx
const ref = useRef<HTMLDivElement>(null);
useScrollReveal(ref as React.RefObject<HTMLElement>, { stagger: 0.13 });
// Options: y, x, duration, stagger, delay, ease, start
```

### `useCountUp(target, suffix, triggerRef, duration)`
GSAP counter: 0 → target when section enters viewport.
```tsx
const numRef = useCountUp(500, "+", triggerRef, 2.0);
<span ref={numRef}>500+</span>
```

### `useCarousel(total, intervalMs)`
Returns: `{ index, dir, next, prev, goTo, onDragEnd, reset, sectionRef }`  
Auto-advances every `intervalMs`. Arrow-key navigation gated by IntersectionObserver (only when section is 30% in view). Pass `0` as interval to pause (e.g. when lightbox is open).

### Shared Stats Config (`src/lib/statsConfig.ts`)
Extracted `STATS_META` constant used by both `About.tsx` and `Results.tsx`. Import from here instead of defining locally.

## ⚠️ Known Pitfalls — Do Not Repeat

| Pitfall | Fix |
|---|---|
| `<line>` JSX in @react-three/fiber | Use `<primitive object={new THREE.Line(...)} />` — `<line>` maps to SVG |
| Duplicate `style=` prop on same element | Merge into one `style={{ a, b }}` object |
| `next.config.ts` fails to load | Use `next.config.js` (plain JS) — Next.js 16 TS transpile bug |
| Framer Motion `ease: "easeOut"` in Variants type error | Use bezier array `[0.25, 0.1, 0.25, 1]` or `import type { Variants }` |
| Adding `mb-*`/`mt-*` to `.layout-container` direct children | Don't — `--vgap` CSS rule handles all spacing; manual margins break symmetry |
| Accessing i18n keys not in `en.json` / `hi.json` | Both files must be updated together for every new key |
| `user-select: none` on `*` in globals.css | **Critical bug** — breaks text selection site-wide. Use `.no-select` utility class instead |
| Hardcoding `md:top-28` for sticky elements | Use `style={{ top: "var(--navbar-h)" }}` to track navbar height |
